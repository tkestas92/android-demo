4. Demo stack'as (vienas app)

docker-compose.yml

  • redroid — Android 13 x86_64, 720×1600, DPI 280, Pixel 9 props, ./redroid-data:/data
  • ws-scrcpy — ghostry/ws-scrcpy, custom index.html mount

ADB

  • connect-adb.sh: adb connect redroid:5555
  • watch-adb.sh kas 3 min (cron) + @reboot start-demo.sh

index.html

  • Auto stream redirect (#!action=stream, player=mse)
  • POST /demo-reset fone (sesijos valymas)
  • Redirect iškart — ne laukiant reset (kitaip rodo device list)

Sesijos reset

pm clear + app restart → login. demo-reset-server.py pagal Host header.
